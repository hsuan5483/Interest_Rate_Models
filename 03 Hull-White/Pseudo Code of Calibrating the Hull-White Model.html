Pseudo Code of Calibrating the Hull-White Model

Author: 許沛萱 Pei-Hsuan Hsu
Email: 07350003@gm.scu.edu.tw

定義
marketR: 市場spot rate
modelR: Hull-White 算出的 spot rate
YC_DF: Yield Curve data

Part 1: 處理資料

Calculating log price of zero-coupon bond

Function
INPUT: Maturity TT, Spot Rate at time t
OUTPUT: lnP(t,T)lnP(t,T)
EndFunction

Calculating instantaneous forward rate

Function
INPUT: Maturity TT, Spot Rate at time tt
OUTPUT: f(t,T)f(t,T)
EndFunction

Calculating the derivative of instantaneous forward rate

Function
INPUT: Maturity TT, Spot Rate at time tt
OUTPUT: ft(t,T)ft(t,T)
EndFunction

Generating information needed in Hull-White model

Function
INPUT: Yield Curve, tenors
OUTPUT: Dataframe YC_DF with [Tenors,marketR,P(t,T),F(0;t,T),f(t,T),ft(t,T)][Tenors,marketR,P(t,T),F(0;t,T),f(t,T),ft(t,T)]
EndFunction

Part 2: The Hull-White Model

Calculating theta in Hull-White model

Function
INPUT: a, sigma, t, YC_DF
OUTPUT: θ(t)θ(t)
EndFunction

Calculating B(t,T)B(t,T)

Function
INPUT: a, sigma, t, T
OUTPUT: B(t,T)B(t,T)
EndFunction

Calculating lnA(t,T)lnA(t,T)

Function
INPUT: a, sigma, t, T, YC_DF
OUTPUT: lnA(t,T)lnA(t,T)
EndFunction

Zero-coupon bond prices under Hull-White model

Function
INPUT: a, sigma, t, YC_DF
OUTPUT: PHull−White(t,T)PHull−White(t,T), RHull−White(t,T)RHull−White(t,T), DF_HullWhite
EndFunction

註：DF_HullWhite為包含Hull-White資訊的dataframe

Objective function for calibrating Hull-White parameters

Function
INPUT: a, sigma, t, YC_DF
OUTPUT: sum of squared error between marketR and RHull−White(t,T)RHull−White(t,T)
EndFunction

Finding optimal parameters by minimize the sum of squared error

Function
INPUT: initial guess of aa
OUTPUT: Optimal Results
EndFunction
